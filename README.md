手机远控软件

  Droidjack是一款针对安卓手机远程管理工具，你可以利用它在PC上对手机进行远程操控，不仅功能强大，使用也很方便。

 - 注意:需要安装java环境
 - 破解版打开弹出输入用户名和密码的框，随便输入即可。

### 功能特性

 - 1、可以生成一个APK，绑定在被控手机的任何APP上
 - 2、可在电脑端控制手机，包括浏览、传输、删除文件等
 - 3、可进行SMS短信收发和查看功能
 - 4、可以控制手机的电话功能
 - 5、联系人管理
 - 6、麦克风监听
 - 7、GPS定位
 - 8、APP管理
